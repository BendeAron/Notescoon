# Notescoon

Simple note-taking UI built with React + Vite.

## Description

Notescoon is a small React project for practicing component-based UI development. It includes a navbar, an “add note” button, and a modal form for creating a note. At the moment, the notes list is static placeholder content and the “Save Note” action doesn’t persist notes yet (no backend/localStorage).

## Getting Started

### Dependencies

* Node.js (LTS recommended) and npm
* Any modern OS (Windows 10/11, macOS, or Linux)

### Installing

* Download the repo (ZIP) or clone it.
* Install dependencies:

```
npm install
```

### Executing program

* Run the development server:

```
npm run dev
```

* Open the URL printed in the terminal (typically `http://localhost:5173/`).
* Build a production bundle:

```
npm run build
```

* Preview the production build locally:

```
npm run preview
```

## Help

If you run into issues:

* If Vite complains about your Node version, upgrade to the latest Node LTS.
* If the default port is busy, Vite will choose another one—check the terminal output.
* If installs get weird, delete `node_modules` and re-run `npm install`.

```
npm run lint
```
